# AutoSeCollege
 
SEカレッジの動画機能がつかいにくいから、スクレイピングツール使って自動化しよう。


# 機能

1. 自動再生

自動で再生して次の動画も自動でクリック
あと、再生速度を1.5倍する。

videoタグが難易度高すぎたので断念。
とりあえず動画が再生するところまでは実装。

1. ランダム再生

ホームページからランダムで選んで表示するように設定。小画面を閉じると自動で次の動画をランダム表示

1. リンク保存機能？

動画のリンクとかをどっかに保存しとくと後で便利かもしれん。

1. 動画ダウンロード機能？

ない。
